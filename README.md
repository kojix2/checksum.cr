# checksum

My personal tool to show md5sum -c or sha256sum -c a little more prettily

## Installation

```sh
git clone https://github.com/kojix2/checksum.cr
cd checksum
shards build --release
cp bin/checksum /usr/local/bin/
```

## Usage

```sh
checksum -c md5sum.txt
```
